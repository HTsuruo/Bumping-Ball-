# Bumping Ball+

空き時間があったら随時更新していく予定.

## Env.
* Xcode 7.3.1
* Swift 2.2

## FrameWork.
* SpriteKit
* GameKit
	* gamecenter, Rank
* Social.framework
	* SNS share (line, twtiter, facebook).
* Multipeer Connectivity Framework
	* bluetooth.
* Fabric 1.6.7

## Libraries.
* Crashlytics 3.8.1
	* crash report.
* NVActivityIndicatorView 2.10
	* loading animation.
* Firebase 3.5.2
* FirebaseMessaging 1.1.1
	* push notification.
* Spring 1.0.3
	* animation.

## Coding Guide.
* [swift-style-guide](https://github.com/github/swift-style-guide)に準拠
* [SwiftLint](https://github.com/realm/SwiftLint)の導入
